Numeros extras permitem que voce envie mensagens a partir de multiplos numeros de WhatsApp, alem do numero principal da sua conta. Cada numero extra e vinculado via codigo de pareamento.

## Cadastro pelo Painel

A forma mais simples de adicionar um numero extra e pelo painel do Turbo Notify. Acesse a pagina de <DashboardLink path="/extra-numbers">Numeros Extra</DashboardLink> e clique em **Adicionar numero**.

Voce devera informar o numero de telefone com codigo do pais e aguardar o codigo de pareamento que sera exibido na tela. Insira esse codigo no WhatsApp do aparelho correspondente para concluir a vinculacao.

## Cadastro via API

## Usabilidade

- Cada numero extra deve pertencer a um aparelho com WhatsApp ativo.
- O codigo de pareamento tem validade limitada. Caso expire, sera necessario refazer a solicitacao.
- Voce pode definir um alias para o numero extra durante o cadastro, facilitando o uso no envio de mensagens via campo `from`.
- Apos o pareamento, o numero ficara com status `pending` ate que a vinculacao seja confirmada.

## Status

Apos cadastrar o numero extra, acompanhe o andamento da vinculacao consultando o [Status dos Numeros Extra](/extra-numbers/status/).

## Erros possiveis

Veja os principais erros que podem ocorrer no cadastro de numeros extras:

| Status | Codigo de erro           | Descricao                                              |
| :----: | ------------------------ | ------------------------------------------------------ |
| `400`  | `invalid_json`           | O JSON da requisicao esta malformado ou invalido.      |
| `402`  | `not_available_in_plan`  | O plano atual nao suporta numeros extras.              |
| `403`  | `not_authorized`         | A chave de acesso esta invalida ou ausente.            |
| `409`  | `number_already_exists`  | O numero informado ja esta cadastrado na conta.        |
| `500`  | `internal_error`         | Erro interno inesperado no servidor.                   |
| `503`  | `whatsapp_unavailable`   | O WhatsApp esta temporariamente fora do ar.            |
