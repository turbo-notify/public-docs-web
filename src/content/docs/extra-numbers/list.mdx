Liste todos os numeros extras vinculados a sua conta para consultar alias, numero de telefone e status de cada um.

## Listar numeros extras

<Tabs>
  <TabItem label="cURL">
    ```bash title="turbo-notify.sh"
    curl -X GET \
      https://api.turbonotify.com/extra-numbers \
      -H "Authorization: Bearer SUA_CHAVE"
    ```
  </TabItem>
  <TabItem label="Node">
    ```js title="turbo-notify.js"
    const response = await fetch('https://api.turbonotify.com/extra-numbers', {
      method: 'GET',
      headers: {
        'Authorization': 'Bearer SUA_CHAVE'
      }
    });

    const result = await response.json();
    console.log(result);
    ```
  </TabItem>
  <TabItem label="Python">
    ```python title="turbo_notify.py"

    url = "https://api.turbonotify.com/extra-numbers"

        "Authorization": "Bearer SUA_CHAVE"

    print(result)
    ```
  </TabItem>
  <TabItem label="Go">
    ```go title="turbo_notify.go"
    package main

    import (
        "fmt"
        "io"
        "net/http"
    )

    func main() {
        req, _ := http.NewRequest("GET", "https://api.turbonotify.com/extra-numbers", nil)
        req.Header.Set("Authorization", "Bearer SUA_CHAVE")
        resp, _ := http.DefaultClient.Do(req)
        defer resp.Body.Close()
        body, _ := io.ReadAll(resp.Body)
        fmt.Println(string(body))
    }
    ```
  </TabItem>
  <TabItem label="PHP">

  <TabItem label="Java">
    ```java title="TurboNotify.java"
    import java.net.http.*;
    import java.net.URI;

    public class TurboNotify {
        public static void main(String[] args) throws Exception {
            var client = HttpClient.newHttpClient();

            var request = HttpRequest.newBuilder()
                .uri(URI.create("https://api.turbonotify.com/extra-numbers"))
                .header("Authorization", "Bearer SUA_CHAVE")
                .GET()
                .build();

            var response = client.send(request,
                HttpResponse.BodyHandlers.ofString());
            System.out.println(response.body());
        }
    }
    ```

## Erros possiveis

| Status | Codigo de erro          | Descricao                                           |
| :----: | ----------------------- | --------------------------------------------------- |
| `402`  | `not_available_in_plan` | O plano atual nao suporta numeros extras.           |
| `403`  | `not_authorized`        | A chave de acesso esta invalida ou ausente.         |
| `500`  | `internal_error`        | Erro interno inesperado no servidor.                |
