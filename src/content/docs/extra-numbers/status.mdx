Acompanhe o status de cada numero extra cadastrado na sua conta para garantir que estejam ativos e prontos para envio.

## Cobranca

A cobranca de numeros extras e feita de forma proporcional ao periodo de uso. Para entender os valores e o ciclo de faturamento, consulte a documentacao de [Cobranca de Numeros Extra](/extra-numbers/billing/).

## Consultar status

Para verificar o status de um numero extra especifico, faca uma requisicao GET informando o alias do numero.

## Tabela de status

| Status     | Descricao                                                        |
| ---------- | ---------------------------------------------------------------- |
| `active`   | O numero esta vinculado e pronto para envio de mensagens.        |
| `inactive` | O numero foi desativado manualmente e nao pode enviar mensagens. |
| `pending`  | A vinculacao esta em andamento, aguardando pareamento.           |
| `failed`   | A vinculacao falhou. Verifique o motivo abaixo.                  |

## Motivos de falha

Quando o status e `failed`, o campo `reason` indica o motivo:

| Motivo                  | Descricao                                                     |
| ----------------------- | ------------------------------------------------------------- |
| `invalid_code`          | O codigo de pareamento inserido esta incorreto ou expirado.   |
| `whatsapp_unavailable`  | O WhatsApp esta temporariamente fora do ar.                   |

## Erros possiveis

| Status | Codigo de erro          | Descricao                                           |
| :----: | ----------------------- | --------------------------------------------------- |
| `402`  | `not_available_in_plan` | O plano atual nao suporta numeros extras.           |
| `403`  | `not_authorized`        | A chave de acesso esta invalida ou ausente.         |
| `404`  | `not_found`             | O alias informado nao foi encontrado na conta.      |
| `500`  | `internal_error`        | Erro interno inesperado no servidor.                |
