Controle quais numeros extras estao ativos para envio de mensagens. Numeros desativados permanecem cadastrados na conta, mas nao podem ser usados como remetente.

## Cobranca

Numeros extras desativados continuam sendo cobrados enquanto estiverem cadastrados. Para encerrar a cobranca, e necessario remover o numero. Consulte os detalhes em [Cobranca de Numeros Extra](/extra-numbers/billing/).

## Pelo Painel

Acesse a pagina de <DashboardLink path="/extra-numbers">Numeros Extra</DashboardLink> no painel para ativar ou desativar numeros com um clique.

## Ativacao

Para ativar um numero extra que esta inativo, faca uma requisicao POST para o endpoint de ativacao.

<Tabs>
  <TabItem label="cURL">
    ```bash title="turbo-notify.sh"
    curl -X POST \
      https://api.turbonotify.com/extra-numbers/activate \
      -H "Authorization: Bearer SUA_CHAVE" \
      -H "Content-Type: application/json" \
      -d '{
        "alias": "clinica_tal"
      }'
    ```
  </TabItem>
  <TabItem label="Node">
    ```js title="turbo-notify.js"
    const response = await fetch('https://api.turbonotify.com/extra-numbers/activate', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer SUA_CHAVE',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        alias: 'clinica_tal'
      })
    });

    const result = await response.json();
    console.log(result);
    ```
  </TabItem>
  <TabItem label="Python">
    ```python title="turbo_notify.py"

    url = "https://api.turbonotify.com/extra-numbers/activate"

        "Authorization": "Bearer SUA_CHAVE",
        "Content-Type": "application/json"

    data = {
        "alias": "clinica_tal"

    response = requests.post(url, json=data, headers=headers)
    result = response.json()
    print(result)
    ```
  </TabItem>
  <TabItem label="Go">
    ```go title="turbo_notify.go"
    package main

    import (
        "fmt"
        "io"
        "net/http"
        "strings"
    )

    func main() {
        payload := strings.NewReader(`{"alias": "clinica_tal"}`)
        req, _ := http.NewRequest("POST", "https://api.turbonotify.com/extra-numbers/activate", payload)
        req.Header.Set("Authorization", "Bearer SUA_CHAVE")
        req.Header.Set("Content-Type", "application/json")
        resp, _ := http.DefaultClient.Do(req)
        defer resp.Body.Close()
        body, _ := io.ReadAll(resp.Body)
        fmt.Println(string(body))
    }
    ```
  </TabItem>
  <TabItem label="PHP">

  <TabItem label="C#">
    ```csharp title="TurboNotify.cs"

    using System.Text.Json;

    var client = new HttpClient();
    client.DefaultRequestHeaders.Add("Authorization", "Bearer SUA_CHAVE");

    var data = new {
        alias = "clinica_tal"
    };

    var json = JsonSerializer.Serialize(data);
    var content = new StringContent(json, Encoding.UTF8, "application/json");

    var response = await client.PostAsync(
        "https://api.turbonotify.com/extra-numbers/activate",
        content
    );
    var result = await response.Content.ReadAsStringAsync();
    Console.WriteLine(result);
    ```
  </TabItem>
  <TabItem label="Java">
    ```java title="TurboNotify.java"
    import java.net.http.*;
    import java.net.URI;

    public class TurboNotify {
        public static void main(String[] args) throws Exception {
            var client = HttpClient.newHttpClient();

            var json = """
                {
                    "alias": "clinica_tal"
                }
                """;

            var request = HttpRequest.newBuilder()
                .uri(URI.create("https://api.turbonotify.com/extra-numbers/activate"))
                .header("Authorization", "Bearer SUA_CHAVE")
                .header("Content-Type", "application/json")
                .POST(HttpRequest.BodyPublishers.ofString(json))
                .build();

            var response = client.send(request,
                HttpResponse.BodyHandlers.ofString());
            System.out.println(response.body());
        }
    }
    ```

## Erros possiveis

| Status | Codigo de erro     | Descricao                                                     |
| :----: | ------------------ | ------------------------------------------------------------- |
| `400`  | `invalid_json`     | O JSON da requisicao esta malformado ou invalido.             |
| `403`  | `not_authorized`   | A chave de acesso esta invalida ou ausente.                   |
| `409`  | `invalid_status`   | O numero ja esta no status solicitado.                        |
| `422`  | `number_not_found` | O alias informado nao corresponde a nenhum numero cadastrado. |
| `500`  | `internal_error`   | Erro interno inesperado no servidor.                          |
