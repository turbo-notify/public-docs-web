Com o Turbo Notify, você envia notificações pelo WhatsApp de forma rápida e fácil. A integração é simples, a API é direta e o <DashboardLink path="/">dashboard</DashboardLink> permite gerenciar tudo em poucos cliques.

Veja como é simples começar:

## Criação da conta

O primeiro passo é criar uma conta gratuita. Basta acessar [Criar Conta Grátis](https://app.turbonotify.com/register) e seguir as instruções. Caso já tenha uma conta, avance para a próxima etapa.

## Chave de acesso

Com a conta criada, acesse o dashboard e toque no menu <DashboardLink path="/access-keys">Chaves de Acesso</DashboardLink>. Gere uma nova chave e copie o valor gerado. Se você já possui uma chave, pode utilizá-la normalmente.

## Primeira notificação

Agora é só enviar sua primeira notificação! Veja abaixo exemplos práticos de envio em diferentes linguagens e tecnologias.
Não se esqueça de substituir o valor da `SUA_CHAVE`. Para detalhes sobre autenticação, consulte a documentação [Chave de Acesso](/general/access-key/).

  <TabItem label="Java">
    ```java title="TurboNotify.java"
    import java.net.http.*;
    import java.net.URI;

    public class TurboNotify {
        public static void main(String[] args) throws Exception {
            var client = HttpClient.newHttpClient();

            var json = """
                {
                    "to": {"number": "+5511999999999"},
                    "content": {"content_type": "text", "text": "Seu pedido foi confirmado!"}
                }
                """;

            var request = HttpRequest.newBuilder()
                .uri(URI.create("https://api.turbonotify.com/v1/messages"))
                .header("Authorization", "Bearer SUA_CHAVE")
                .header("Content-Type", "application/json")
                .POST(HttpRequest.BodyPublishers.ofString(json))
                .build();

            var response = client.send(request,
                HttpResponse.BodyHandlers.ofString());
            System.out.println(response.body());
        }
    }
    ```
