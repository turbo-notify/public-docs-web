Com o Turbo Notify, voce envia notificacoes pelo WhatsApp de forma rapida e facil. A integracao e simples, a API e direta e o <DashboardLink path="/">dashboard</DashboardLink> permite gerenciar tudo em poucos cliques.

Veja como e simples comecar:

## Criacao da conta

O primeiro passo e criar uma conta gratuita. Basta acessar [Criar Conta Gratis](https://app.turbonotify.com/register) e seguir as instrucoes. Caso ja tenha uma conta, avance para a proxima etapa.

## Chave de acesso

Com a conta criada, acesse o dashboard e toque no menu <DashboardLink path="/access-keys">Chaves de Acesso</DashboardLink>. Gere uma nova chave e copie o valor gerado. Se voce ja possui uma chave, pode utiliza-la normalmente.

## Primeira notificacao

Agora e so enviar sua primeira notificacao! Veja abaixo exemplos praticos de envio em diferentes linguagens e tecnologias.
Nao se esqueca de substituir o valor da `SUA_CHAVE`. Para detalhes sobre autenticacao, consulte a documentacao [Chave de Acesso](/general/access-key/).

    package main

    import (
        "fmt"
        "io"
        "net/http"

    func main() {
        payload := strings.NewReader(`{"to": "+5511999999999", "text": "Seu pedido foi confirmado!"}`)

    var client = new HttpClient();

  <TabItem label="Java">
    ```java title="TurboNotify.java"
    import java.net.http.*;
    import java.net.URI;

    public class TurboNotify {
        public static void main(String[] args) throws Exception {
            var client = HttpClient.newHttpClient();

            var json = """
                {
                    "to": "+5511999999999",
                    "text": "Seu pedido foi confirmado!"
                }
                """;

            var request = HttpRequest.newBuilder()
                .uri(URI.create("https://api.turbonotify.com/messages"))
                .header("Authorization", "Bearer SUA_CHAVE")
                .header("Content-Type", "application/json")
                .POST(HttpRequest.BodyPublishers.ofString(json))
                .build();

            var response = client.send(request,
                HttpResponse.BodyHandlers.ofString());
            System.out.println(response.body());
        }
    }
    ```
